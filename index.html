CAPABILITIES

Everything you need to audit agent trades

monitoring

Trade Reconstruction

Automatically match entries and exits, reconstruct complete trade lifecycles from raw order fills.

shield

Risk Scoring

Composite risk score from drawdown, leverage exposure, concentration, and overtrading signals.

Policy Firewall

Pre-trade intent evaluation against configurable rules - symbol allow-lists, leverage caps, position limits.

CLI & Web UI

WORKFLOW

Three steps. Zero key exposure.

01

Export Logs Locally

Run bitguard collect-bitget with your own read-only Bitget credentials. Keys stay on your machine.

02

Run the Audit

Use bitguard audit or paste JSON into the web dashboard. Trades are reconstructed and scored instantly.

03

Review & Share

View findings in the dashboard, redact sensitive data with bitguard redact, then share the clean report.

$ pip install -r requirements.txt
